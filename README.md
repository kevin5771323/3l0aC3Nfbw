# 项目简介

## 核心功能
- 基于YOLOv8的行人车辆密集检测系统
- 支持图片、视频及摄像头进行检测
- 支持选择模型
- 界面实时显示目标位置、目标总数、置信度等信息
- 支持批量检测并直接查看所有检测结果
- 支持检测结果保存
- 自定义置信度和IOU阈值
- 可选择显示标签和原图
- 提供多种检测模型选择

## 适用场景
- 交通监控系统
- 安防监控系统
- 车辆流量分析
- 行人流量统计
- 智能停车场管理
- 城市安全监控

## 技术特点
- 基于YOLOv8深度学习框架
- 使用Pycharm/VSCode开发环境
- 依赖Python 3.9、OpenCV、PyQt5、Ultralytics、Torch1.9等技术栈
- 内置3000张VOC2007数据集，包含7类目标检测对象
- 提供完整标注的数据集和训练好的模型
- 包含项目运行教程和详细文档
- 配置成功环境后可直接使用
